Sobre o Gamory

O Gamory é uma plataforma desenvolvida para amantes de videogames. Aqui, jogadores podem avaliar, comentar e compartilhar suas experiências com jogos de todos os tipos. Seja para lembrar do que já jogou ou descobrir novos títulos, o Gamory é seu espaço para viver o mundo gamer de forma mais conectada.

Objetivos

  • Facilitar a troca de opiniões entre jogadores.
  • Ajudar usuários a descobrirem novos jogos.
  • Criar um histórico pessoal de jogos jogados e avaliados.
  • Fomentar uma comunidade saudável e crítica.

Recursos

  • Avaliações com estrelas
  • Resenhas e comentários
  • Busca de jogos
  • Perfis de usuários

Tecnologias usadas

  • HTML, CSS e JavaScript puro foram utilizados para criar uma interface clara, responsiva e interativa. Com animações, efeitos visuais e um layout intuitivo, buscamos uma experiência imersiva para os jogadores.

    Utilizamos Node.js com o framework Express.js para estruturar o servidor, lidar com rotas, autenticação e comunicação com o banco de dados.

    Usamos MySQL para armazenar informações dos usuários, avaliações, resenhas e dados dos jogos. As queries são otimizadas para garantir desempenho e segurança.

    Integração com API: A plataforma se conecta com a IGDB API (Internet Game Database), permitindo a exibição de informações atualizadas sobre os jogos como nome, imagem, gênero e descrição.
